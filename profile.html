HTML as One of My Core Skills

HTML stands for HyperText Markup Language. It is the standard language used to create and structure content on the web. HTML is not a programming language — it is a markup language, meaning it uses tags to label and describe pieces of content so the browser knows how to display them.

1. Structural Elements

<html>

What it is: The root element of every HTML page.

Used for: Wrapping all content on the page.

When to use: Always — every HTML file must have this tag.

Example:

<html lang="en"> ... </html>

<head>

What it is: A container for metadata about the page.

Used for: Holding the title, character set, and links to stylesheets.

When to use: Always — placed before the body tag.

Example:

<head> <title>My Page</title> </head>

<body>

What it is: Contains all visible content on the page.

Used for: Everything the user sees — text, images, links, tables.

When to use: Always — all visible content goes inside here.

Example:

<body> <h1>Hello</h1> </body>

<header>

What it is: A semantic tag for the top section of a page or section.

Used for: Introductory content like a name, title, or logo.

When to use: At the top of your page or inside a section.

Example:

<header> <h1>Moses Ninah</h1> </header>

<nav>

What it is: A semantic tag for navigation links.

Used for: Grouping links that help users move around the page.

When to use: When you have a set of links for navigation.

Example:

<nav> <a href="#about">About</a> </nav>

<section>

What it is: A semantic tag that defines a standalone section of content.

Used for: Grouping related content together with a clear purpose.

When to use: When a block of content forms its own topic or chapter.

Example:

<section id="about"> ... </section>

<footer>

What it is: A semantic tag for the bottom section of a page.

Used for: Copyright info, contact details, or closing notes.

When to use: At the bottom of the page.

Example:

<footer> <p>© 2026 Moses Ninah</p> </footer>

2. Text Formatting Elements

<h1> to <h6>

What it is: Heading tags from largest (h1) to smallest (h6).

Used for: Titles and subtitles to organize content hierarchy.

When to use: h1 for the main title, h2 for section titles, h3 for subtopics.

Example:

<h1>Main Title</h1>
<h2>Section Title</h2>
<h3>Subtopic</h3>

<p>

What it is: A paragraph tag.

Used for: Blocks of text and general written content.

When to use: Whenever you write a sentence or paragraph.

Example:

<p>I am a cybersecurity student.</p>

<strong>

What it is: Makes text bold and marks it as important.

Used for: Emphasizing important words.

When to use: When a word or phrase is critically important.

Example:

<strong>Important</strong>

<em>

What it is: Makes text italic and marks it as emphasized.

Used for: Stressing a word in a sentence.

When to use: When you want to stress or emphasize a word.

Example:

<em>This is important</em>

<br>

What it is: A line break tag — it has no closing tag.

Used for: Moving content to the next line.

When to use: When you need a new line without starting a new paragraph.

Example:

Moses<br>Ninah

<hr>

What it is: A horizontal rule — draws a line across the page.

Used for: Visually separating sections of content.

When to use: Between major sections to create a clear visual break.

Example:

<hr>

<pre>

What it is: Displays text exactly as written, preserving spaces and line breaks.

Used for: Showing code examples or formatted text.

When to use: When displaying code or text that needs exact formatting.

Example:

<pre>  This    is    spaced</pre>

3. Lists

<ul> — Unordered List

What it is: A bulleted list with no particular order.

Used for: Items where order does not matter.

When to use: For interests, features, or general lists.

Example:

<ul>
  <li>Cybersecurity</li>
  <li>Rugby</li>
</ul>

<ol> — Ordered List

What it is: A numbered list where order matters.

Used for: Steps, rankings, or sequences.

When to use: When the order of items is important.

Example:

<ol>
  <li>Step One</li>
  <li>Step Two</li>
</ol>

<li> — List Item

What it is: A single item inside a list.

Used for: Each individual entry in a ul or ol.

When to use: Always inside a ul or ol — never on its own.

4. Links

<a>

What it is: An anchor tag — creates a clickable hyperlink.

Used for: Linking to other pages, websites, or sections on the same page.

When to use: Whenever you need a clickable link.

Example:

<a href="https://google.com">Visit Google</a>
<a href="#about">Go to About Section</a>

5. Images

<img>

What it is: An image tag — it has no closing tag.

Used for: Displaying images on a page.

When to use: When you want to show a photo or graphic.

Example:

<img src="photo.jpg" alt="A photo of Moses">

The src attribute is the path to the image. The alt attribute is a text description shown if the image fails to load.

6. Tables

<table>, <tr>, <th>, <td>

What it is: A set of tags that together create a table.

Used for: Displaying data in rows and columns.

When to use: When presenting structured data like facts, comparisons, or records.

Example:

<table border="1">
  <tr>
    <th>Name</th>
    <th>Course</th>
  </tr>
  <tr>
    <td>Moses</td>
    <td>Cybersecurity</td>
  </tr>
</table>

tr = table row. th = table header (bold). td = table data (normal cell).

7. Forms

<form>

What it is: A container for collecting user input.

Used for: Login pages, contact forms, search boxes.

When to use: When you need to collect information from the user.

<input>

What it is: A field where the user can type or select something.

Used for: Text fields, checkboxes, radio buttons, submit buttons.

When to use: Inside a form to collect a specific piece of input.

Example:

<form>
  <label>Name:</label>
  <input type="text" name="name">
  <br>
  <input type="submit" value="Submit">
</form>

<label>

What it is: A label that describes an input field.

Used for: Telling the user what to type into a field.

When to use: Always paired with an input field.

8. Semantic Elements

Semantic elements are tags that carry meaning — they tell the browser and developer what kind of content is inside, not just how it looks.

Tag Meaning Used For
<header> Top of the page or section Page title, logo, intro
<nav> Navigation Links to move around the page
<section> A standalone topic About Me, Interests, Skills
<article> Independent content Blog posts, news items
<aside> Side content Extra info, sidebars
<footer> Bottom of the page Copyright, closing info
<main> Main content area The primary content of the page
